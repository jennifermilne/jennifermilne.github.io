Assignment 1 Final Model

What is this model?

This model takes a number of agents (specified by the user) and places them in an environment where
they move randomly and eat the environment that they have touched. They move up to the number of
iterations set. The environment scale can also be changed to whatever is wanted.

Information for running the model

Open model10.py and go to
Tools > Preferences > iPython Console > Graphics... and set the Backend to Tkinter
